## 技术栈

### 前端
1. HTML
2. CSS
3. Thymeleaf
4. jQuery
### 后端
1. SpringMVC
2. SpringBoot
3. SpringSecurity
4. MyBatis
5. MySQL
6. Redis
7. Swagger

2. maven版本：maven3.0以上

3. 数据库: MySQL8.0
